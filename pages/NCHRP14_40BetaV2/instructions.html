Instructions

  1. Enter Project Information in colored boxes at top of page.
    • It is recommended that you save your inputs at this point using the green Save Button at the top of the page so as to avoid accidentally saving your changes to the template file.
  2. Answer all questions in the Project Questionnaire section that have colored boxes for the answer. (You can navigate directly to each entry by pressing the Tab key on your keyboard)
    • Please Note that questions 6 & 7 require 2 entries - the numeric value and appropriate units.
    • If unsure how to answer the question, please click on the blue button to "Show Input Hints" at the top of the page. These hints can be toggled on and off by clicking the button again.
  3. For each Core Mix to be used (up to 3), answer all questions that have colored boxes for the answer.
    • Please Note that questions 2a-2e & 3a-3b require 2 entries - the numeric value and appropriate units & watch for any warning messages that may pop-up or be displayed in red to the right of the questions
    • The number of mixes to be used is entered under the Project Questionnaire section & should be discussed with the District Roadside Manager if you are unsure of the answer.
    • When answering questions 2c or 2e about the steepest gradients, the default "units" are set to be entered as a slope ratio (i.e. 4:1 h:v), but only the first part of the ratio should be entered. For example, to enter a 3:1 slope, type in just 3, and the program will automatically show 3.0:1. Fractional slopes (such as 7:2) may be entered as a formula (=7/2) or a decimal (3.5) and will automatically be converted to a ":1" ratio. Alternatively, the units can be changed to "Grade" and entered as a percentage (i.e. 40% or 0.4)
    • For any question in these sections that require direct entry of areas with units, an area calculator tool is provided in case you do not already know the size of the area in either square feet or acres. It can be used for unit conversions (metric to imperial, yards or miles to feet, etc.) or combining multiple sub-areas into one total area for each core mix.
    • To use the area calculator tool, click on the button beside question 2a to open it and:
      1. Fill out either Length & Width or Area and select the proper units for each entry.
        • If sub areas are repeated (for example, 3 areas with the same length/width), they can be entered once by setting the multiplier at the bottom of the sub-area to the number of times the shape is repeated.
      2. Answer the question about the maximum slope of each area.
        • Note: areas should be subdivided based on a break point of 3:1 slopes (33.33% grade) in order to assign the areas as Mow or Non-Mow correctly.
      3. Once all sub-areas are entered, press the green button to "Use Results on Start Page."
  4. It is recommended that you save your work after answering the questions for each section using the green Save Button at the top of the page prior to continuing.
  5. Once all colored boxes in each section have been filled in, click the green "Show Results" button.

Fertilizer Page (if necessary, i.e. not using default fertilizer rates)

  1. Ensure that the Testing Laboratory selected & it's corresponding Test Method is correct.
  2. Answer the question about recommendations provided by the testing lab (below the Project Info section)
      └» If the answer is yes, enter the recommended application rate in tons/ac based on Agricultural Lime
    • Note: if the rate provided was not in tons/acre, you can use excel to convert the rate. The following conversion factors may be useful:
      1 acre = 43,560 square feet; 1 sq. yd. = 9 sq. ft.; 1 sq. mile = 27,878,400 sq. ft.
      1 sq. km = 100 hectares; 1 hectare = 107,639 sq. ft.; 1 sq. meter = 10.76 sq. ft.
      1 ton [US] = 2,000 lbs; 1 kg = 2.205 lbs; 1 metric ton = 2,204.6 lbs"
  3. Select the Type (Group) of Lime to be used
      └» For Groups B & C, select the % CCE that is closest to your specific product.
  4. Enter the pH Buffer for each sample. Please note that the pH range allowed by each test varies, and the program will only allow values within this range to be entered.
    • You can skip this step if the answer from step #2 above is yes (the lab provided a Lime application rate recommendation either based on an average of all the samples analyzed, or the recommendations for each sample were the same)

Lab Analysis Summary Page (only visible if not using default nutrient rates)

  1. Confirm that there are no errors and all summaries match the information from the previous pages.
  2. If subdividing the areas to be permanently seeded into groups based on similar soil sample results, enter the approx. areas of each subgroup in the Fertilizer Summary area. Otherwise, leave those fields blank or enter 0.

Nutrient Rates, Mulch, and Roadside Summary

  1. Confirm that there are no errors and all summaries match the information from the previous pages.

Roadside Notes

  1. Choose whether or not to include each note by entering "Yes" or "No" in the box to the left of the note.
  2. Enter any project specific Custom Notes at the bottom of the page as desired.

Seed Selection
Note: Seed selection is to be performed by the VDOT District Roadside Manager or his/her delegate!

  1. On the Start Page click the blue button at the bottom of the page to open the Seed Rate Selection Page.
  2. Answer the questions at the top regarding the use of custom seed mixes, & if they are to be used on this project, fill out the sections for each custom seed mix at the bottom of the page.
    (There can be up to 3 custom seed blends to use in place of core seed mixes on each project)
    • Hyperlinks are provided to the right of the questions to navigate directly to the entry for each blend. Instructions can be found beside the input area for the first custom mix.
  3. Adjust Application Rate for Core Mixes & Additives as desired.
  4. If additional additives that are not already on the list are to be used, enter a Description, Application Rate (in lbs/ac) and whether or not the additive is a Legume. If the additive is restricted from use in mowed areas, please indicate that with two asterisks (**) at the end of the description.
  5. Click on the blue button at the top of the page to Open the Seed Mix Selection Page.
    (This button can also be found at the bottom of the start page).
  6. On the Seed Mix Selection page, use the drop down lists to select a Core Mixture for each Season for both Mow & Non-Mow areas (Slopes). The entire top row of each Core Mix used must be filled out.
    • The drop down list can be opened by clicking on the gray button at the right of each colored cell when it is selected
  7. Use the drop down lists to include additives as desired for each season for both Mow & Non-Mow areas
  8. Navigate to the Seed Used page to review the list of seeds & additives and their application rates.
    • If so desired, use the blue button at the top of the page to toggle between showing all seeds available and just the seeds that have been selected for use on this project.
  9. Review the Custom Seed Mix lists at the bottom of the page to ensure that there are no warnings.
    • NOTE: The seed lists and seed mix selection tabs are set to automatically be hidden each time the spreadsheet is closed (so long as the user saves changes after closing the program).

Printing the Report

  1. The report can be printed from any page that has the green "Print Report" button.
      2. NOTE: The results should be printed to PDF so that the Nutrient Summary, Mulch, Seed Lists, Roadside Summary, and Roadside Notes pages can be imported into the Roadside Development sheets in MicroStation.
